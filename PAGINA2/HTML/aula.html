Resumen del curso

En este curso aprenderás los fundamentos básicos para crear sitios web: estructura con HTML, estilos con CSS y una introducción a JavaScript para añadir interacción.

  • Duración: 8 semanas
  • Modalidad: 100% en línea
  • Docente: Ing. Laura Pérez

Unidades del curso

Unidad 1: Fundamentos de la Web
  • Cliente, servidor y HTTP
  • Navegadores y motores de renderizado
  • Arquitectura básica de una aplicación web
Unidad 2: HTML básico
  • Estructura de un documento HTML
  • Encabezados, párrafos, enlaces e imágenes
  • Listas y tablas
Unidad 3: CSS y maquetación
  • Reglas, selectores y propiedades
  • Uso de frameworks CSS (Skeleton)
  • Diseño responsivo con grid
Unidad 4: Introducción a JavaScript
  • Variables y tipos de datos
  • Eventos en el navegador
  • Manipulación básica del DOM
